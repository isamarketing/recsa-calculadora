# RECSA — Calculadora de Descuentos Comerciales

Herramienta interactiva para calcular descuentos a clientes basada en 3 variables ponderadas:
- **Volumen** (kg/semestre) — peso default 50%
- **Portafolio** (cantidad de productos) — peso default 30%
- **Acuerdo** (meses de compromiso) — peso default 20%

## Acceso
https://isamarketing.github.io/recsa-calculadora/

## Uso
- Ajustar pesos de variables con los sliders
- Editar escalas/tiers directamente en las tablas
- Seleccionar productos, volumen y plazo del deal
- El descuento se calcula automáticamente

## Líneas de producto
- Adhesivos (BOND AG 50, RF 7)
- Adhesivos ML — Construcción en seco
- Selladores — Alta performance (SEAL)
- Prenadores (PREN 172)
- Bond HP — Drenaje (Estrella)
- WRC Impermeabilizantes (Estrella)
